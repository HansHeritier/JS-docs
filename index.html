Les Bases :

Déclaration / Initialisation

Fonction : Créer une variable

i

Opérateur :

var
let
const

Syntaxe :

var maVariable
let maVariable
const maVariable

Assignation

Fonction : Donner une valeur à une variable

i

Opérateur :

=

Syntaxe :

var maVariable = /* la valeur de ma variable */

Pop-up

Fonction : Afficher fenêtre une pop up

i

Opérateur :

prompt

Syntaxe :

var maVariable = prompt (“ ici ma question ”)

Affichage Console

Fonction : Afficher dans la console

i

Opérateur :

console.log()

Syntaxe :

console.log (maVariable)

Concaténation

Fonction : Chaîne de caractère contenant différente valeur ajoutée

i

Opérateur :

“ “ + “ “

Syntaxe :

“ La valeur de ce présent est “ + maVariable + “ inestimable“

Interpolation

Chaîne de caractère contenant différente valeur ajoutée

i

Opérateur :

‘ $ {} ’

Syntaxe :

‘ La valeur de ce présent est ${ maVariable } ’

Comparaison

Fonction : Comparer 2 objets ou 2 variables

i

Opérateur :

=== strictement égale
!== différent de
< + petit <= + petit ou =
> + grand >= + grand ou =

Syntaxe :

maVar === maVar

Condition

Fonction : Conditionner une fonction selon les paramètre défini

i

Opérateur :

si if ( ) { }
alors else { }
ou if ( ) { }
else if { }

Syntaxe :

if (condition) { Code à exécuter si renvoi true }
else if (condition) { Code à exécuter}
else { Code à exécuter }

Opérateurs logiques

Fonction : Conditionner ou nuancer une fonction selon plusieurs paramètres

i

Opérateur :

ou ||
et &&

Syntaxe :

conditionA || conditionB
conditionA && conditionB

Loop while

Fonction : Exécuter un code en boucle selon une condition

i

Opérateur :

Tant que while ( ) { }

Syntaxe :

let maVariable /*Expression initiale*/
while (condition à remplir) {
/*code à exécuter*/
/* expr mis à jour (incré ou décré)*/ }

Loop for

Fonction : Exécuter un code en boucle selon une condition

i

Opérateur :

Pour for ( ; ; ) { }

Syntaxe :

for (expression initiale ;
condition ;
expression mis à jour) {
/* code à exécuter */ }
